# macos-setup

Reproducible macOS dev environment for a fresh MacBook.

## Quickstart (new mac)

```sh
# 1. Clone (xcode-select will trigger if git isn't installed yet)
git clone https://github.com/adechassey/macos-setup.git ~/Documents/perso/code/macos-setup
cd ~/Documents/perso/code/macos-setup

# 2. Bootstrap
bash setup.sh

# 3. Reload shell
exec zsh
```

`setup.sh` is idempotent — re-run anytime to sync.

## What's inside

| File | Role |
|---|---|
| `Brewfile` | All formulas + casks (apps), restored via `brew bundle` |
| `setup.sh` | Bootstrap: Xcode CLT → Homebrew → Brewfile → dotfile symlinks → git identity → Node LTS → VS Code extensions |
| `home/` | Mirror of `$HOME`, symlinked into place |
| `home/.zshrc` | Shell config: plugins, aliases (32 git + ls + personal), prompt |
| `home/.zprofile` | Sets up Homebrew env at login |
| `home/.gitconfig` | Generic settings + `[include]` of gitignored `~/.gitconfig.local` |
| `home/.config/ghostty/config` | Snazzy theme |
| `home/.config/gh/config.yml` | gh CLI prefs |

## What's installed

**Shell**: starship, zsh-autosuggestions, zsh-completions, zsh-fast-syntax-highlighting, fzf, zoxide

**CLI tools**: bat, eza, fd, ripgrep, tldr, gh, fnm

**Apps**: Ghostty, VS Code, Docker, Chrome, Raycast, Slack

## What's NOT in the repo

- `~/.gitconfig.local` — your name/email, gitignored. Generated by setup.sh on first run.
- `~/.config/gh/hosts.yml` — gh OAuth state, gitignored.
- SSH keys, AWS creds, etc. Set those up by hand.

## Updating

Edit files in `home/` and commit. Since they're symlinked, edits take effect immediately on the source machine — no re-run needed.

To add a new tool: add it to `Brewfile`, then `brew bundle install`.
